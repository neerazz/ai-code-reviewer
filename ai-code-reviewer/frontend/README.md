# Frontend

This directory contains the React frontend for the AI-Powered Code Review & Migration Assistant.

## Structure

The frontend is structured as follows:

-   `public/`: The public assets.
-   `src/`: The source code.
    -   `components/`: The React components.
-   `Dockerfile`: The Dockerfile for building the frontend image.
-   `package.json`: The Node.js dependencies.
-   `tailwind.config.js`: The Tailwind CSS configuration.
-   `postcss.config.js`: The PostCSS configuration.

## Running Locally

To run the frontend locally, you will need to have Node.js and `npm` installed.

1.  **Install the dependencies:**

    ```bash
    npm install
    ```

2.  **Start the server:**

    ```bash
    npm start
    ```

This will start the development server on [http://localhost:3000](http://localhost:3000).
