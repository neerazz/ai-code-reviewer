# Welcome to the AI Code Reviewer Docs

This documentation site will guide you through the features and usage of the AI Code Reviewer.

## What is the AI Code Reviewer?

The AI Code Reviewer is an intelligent code review system that uses LLMs to provide context-aware code reviews, suggest optimizations, detect security vulnerabilities, and automate code migrations across different frameworks/versions.
